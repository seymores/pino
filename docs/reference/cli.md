# CLI Reference

Commands:
- `clojure -M -m pino.cli.main generate app <name> --target <dir>`
- `clojure -M -m pino.cli.main generate feature <name> --target <dir>`
- `clojure -M -m pino.cli.main generate model <name> --target <dir>`
- `clojure -M -m pino.cli.main generate resource <name> --target <dir>`
- `clojure -M -m pino.cli.main generate auth setup --target <dir>`
- `clojure -M -m pino.cli.main check`
